1. Базовое заполнение нулями (c = 0)

- Описание: Проверка, что функция корректно заполняет память нулями.
- Входные данные:
    - Буфер: "abcdefgh" (размер 8).
    - c = 0, n = 3.
- Ожидаемый результат: Первые 3 байта заменены на 0: {0, 0, 0, 'd', 'e', 'f', 'g', 'h'}.


2.Заполнение ненулевым символом (c = 'A')

- Описание: Проверка заполнения памяти указанным символом.
- Входные данные:
    - Буфер: "abcdefgh" (размер 8).
    - c = 'A', n = 2.
- Ожидаемый результат: Первые 2 байта заменены на 'A': {'A', 'A', 'c', 'd', 'e', 'f', 'g', 'h'}.


3. Полное заполнение буфера (n = размер буфера)
- Описание: Проверка, что функция корректно заполняет весь буфер.
- Входные данные:
    - Буфер: "abcd" (размер 5, включая '\0').
    - c = 'x', n = 5.
- Ожидаемый результат: Весь буфер заполнен 'x': {'x', 'x', 'x', 'x', 'x'}.


4. Заполнение нулевого размера (n = 0)

- Описание: Проверка, что при n = 0 буфер не изменяется.
- Входные данные:
    - Буфер: "test" (размер 5).
    - c = 'y', n = 0.
- Ожидаемый результат: Буфер остался без изменений: "test".


5. Отрицательное значение c (приведение к unsigned char)

- Описание: Проверка, что отрицательный c корректно приводится к unsigned char.
- Входные данные:
    - Буфер: {1, 2, 3, 4, 5} (5 байт).
    - c = -1, n = 3.
- Ожидаемый результат: Первые 3 байта заменены на 255 (0xFF): {255, 255, 255, 4, 5}.


6. Значение c больше 255 (обрезание до c % 256)

- Описание: Проверка, что c обрезается до одного байта.
- Входные данные:
    - Буфер: {1, 2, 3, 4, 5} (5 байт).
    - c = 0x1234, n = 2.
- Ожидаемый результат: Первые 2 байта заменены на 0x34: {0x34, 0x34, 3, 4, 5}.


7. Заполнение int-массива

- Описание: Проверка работы с не-char типами данных.
- Входные данные:
    - Массив: {1, 2, 3} (3 int).
    - c = 0, n = sizeof(arr).
- Ожидаемый результат: Весь массив заполнен нулями: {0, 0, 0}.


8. Заполнение структуры

- Описание: Проверка работы со структурами.
- Входные данные:
    - Структура: {char a = 'x'; int b = 42;}.
    - c = 0, n = sizeof(struct).
- Ожидаемый результат: Вся структура заполнена нулями: {a = 0, b = 0}.


9. Проверка возвращаемого значения

- Описание: Проверка, что функция возвращает указатель на переданный буфер.
- Входные данные:
    - Буфер: char buffer[10].
    - c = 'a', n = 5.
- Ожидаемый результат: Возвращаемое значение == buffer.


10. Передача NULL (неопределённое поведение, UB)

- Описание: Проверка реакции на некорректный аргумент.
- Входные данные:
    s = NULL, c = 0, n = 10.
- Ожидаемый результат:
    Варианты:
        - Падение с SIGSEGV (если система защищена).
        - Ничего не происходит (если реализация игнорирует NULL).


11. Переполнение буфера (n > размер буфера)

- Описание: Проверка обработки выхода за границы буфера.
- Входные данные:
    - Буфер: char small[3].
    - c = 'x', n = 10.
- Ожидаемый результат:
    Варианты:
        - Падение с SIGSEGV (если есть защита памяти).
        - Запись за пределы буфера (UB).
