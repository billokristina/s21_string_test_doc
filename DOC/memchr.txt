1. Базовые сценарии
   - 1.1 Символ присутствует в строке
     - 1.1.1
       - Строка: "hello"
       - Символ: 'e'
       - Размер: 5
       Ожидаемый результат: Указатель на 'e' (второй символ).

   - 1.1.2
     - Строка: "test_string"
     - Символ: '_'
     - Размер: 11
     Ожидаемый результат: Указатель на '_' (пятый символ).

   - 1.1.3
     - Строка: "abcdef"
     - Символ: 'f'
     - Размер: 6
     Ожидаемый результат: Указатель на 'f' (последний символ).

   - 1.2 Символ отсутствует в строке
     - 1.2.1
       - Строка: "hello"
       - Символ: 'x'
       - Размер: 5
       Ожидаемый результат: NULL.
     - 1.2.2
       - Строка: "12345"
- Символ: '0'
- Размер: 5

Ожидаемый результат: NULL.

# 2. Граничные случаи

## 2.1 Символ в начале строки

- Строка: "hello"
- Символ: 'h'
- Размер: 5

Ожидаемый результат: Указатель на начало строки.

## 2.2 Символ в конце строки

- Строка: "world"
- Символ: 'd'
- Размер: 5

Ожидаемый результат: Указатель на последний символ.

## 2.3 Символ за пределами размера n

- Строка: "abcdef"
- Символ: 'f'
- Размер: 5

Ожидаемый результат: NULL (символ 'f' находится за пределами первых 5 байт).

## 2.4 Размер n = 0

- Строка: "hello"
- Символ: 'h'
- Размер: 0

Ожидаемый результат: NULL (поиск не выполняется).

2.5 Символ '\0' (нулевой байт)

- Строка: "abc\0def"
- Символ: '\0'
- Размер: 7

Ожидаемый результат: Указатель на '\0' (четвертый символ).

3. Ошибочные случаи

3.1 Пустая строка

- Строка: ""
- Символ: 'a'
- Размер: 0

Ожидаемый результат: NULL.

3.2 Невалидный указатель (NULL)

- Строка: NULL
- Символ: 'a'
- Размер: 5

Ожидаемый результат: Ошибка (поведение зависит от реализации, но обычно это приводит к краху).

4. Специальные символы и бинарные данные

4.1 Символы вне ASCII

- Строка: "café"
- Символ: 'é'
- Размер: 5

Ожидаемый результат: Указатель на 'é'.

4.2 Бинарные данные

- Строка: "\x01\x02\x03\x04\x05"
- Символ: '\x03'
- Размер: 5

Ожидаемый результат: Указатель на третий байт.

5. Повторяющиеся символы

- Строка: "aabbcc"
- Символ: 'b'
- Размер: 6

Ожидаемый результат: Указатель на первое вхождение 'b'.

6. Большие данные

- Строка: Длинная строка (например, 1 МБ данных).
- Символ: Символ, находящийся в середине строки.
- Размер: Размер строки.

Ожидаемый результат: Указатель на первое вхождение символа.

7. Символы с отрицательными значениями

- Строка: "\xFF\xFE\xFD"
- Символ: '\xFE'
- Размер: 3

Ожидаемый результат: Указатель на второй байт.

8. Символы, выходящие за пределы char

- Строка: "abc"
- Символ: 256 (выходит за пределы char)
- Размер: 3

Ожидаемый результат: NULL (так как символ не может быть найден).
